## 🚀 スキル強化プログラム（2026-05-20 実施）

> 目的：業界リサーチ領域で唯一無二・オーバースペックな調査力を確立する。現状スキルを棚卸しし改善余地を10ステップで埋める。

### STEP 1｜現状スキル棚卸し
できていること：建設業界の採用トレンド調査 / 競合のSNS・LP・求人調査 / 業界ニュース・法改正の収集 / 求職者リサーチ / 他業界事例の横展開。**情報収集は広いが、出典の信頼性評価と分析フレームワーク、定量裏取りが弱い。**

### STEP 2｜業界トップ水準とのギャップ分析
不足：①一次情報と二次情報を区別せず扱っている ②出典の信頼性を階層で評価していない ③数字の裏取り（複数ソース照合）が弱い ④PEST・5Forces等の分析フレームワークが体系化されていない ⑤リサーチ手順（検索クエリ・日付）が記録されず再現性がない。

### STEP 3｜改善余地・成長余地の特定
埋めるべき5項目：出典信頼性の階層評価 / 情報の三角測量（裏取り）/ 分析フレームワークの体系化 / リサーチ手順の標準化 / 示唆の戦略接続。

### STEP 4｜知識補充（リサーチ専門知識）
- **情報の信頼性階層**：政府・公的統計 ＞ 業界団体 ＞ 専門メディア ＞ 一般報道 ＞ 個人発信。下位ほど裏取り必須。
- **一次情報 vs 二次情報**：求職者インタビュー・公的統計は一次。まとめ記事は二次。提案の根拠は一次情報を優先。
- **建設業の構造知識**：2024年問題（時間外労働の上限規制）/ 技能者の高齢化と若手不足 / 有効求人倍率の高止まり / 週休二日・処遇改善の流れ。これらが採用訴求の土台。
- **競合分析の観点**：訴求軸・媒体・更新頻度・見せ方・弱点。「真似る点」でなく「差別化できる空白」を探す。
- **求職者リサーチ**：20代未経験層は「給与の見通し」「人間関係」「成長実感」を重視。不安の言語化が刺さるコピーになる。

### STEP 5｜新規フレームワーク習得
PEST分析（政治・経済・社会・技術）/ ファイブフォース / SWOT / 競合ベンチマークマトリクス / 情報の三角測量（最低2ソースで照合）。

### STEP 6｜出力品質基準の引き上げ
レポートに必須追加：①各情報の**出典と取得日** ②**一次/二次の区別** ③数字は**複数ソースで裏取り** ④示唆に**確度（高/中/低）**と**戦略への接続**。

### STEP 7｜ツール・仕組みの拡充
官公庁統計・業界団体サイトの一次ソース優先 / Google News アラート / 検索クエリと取得日のリサーチログ化 / 競合LPパターン分類の継続。

### STEP 8｜失敗パターンと回避策
①出典不明の数字を引用→必ず一次ソースを当たる ②古い情報を最新と誤認→取得日を明記 ③印象論→PEST・5Forces等の枠で構造化。

### STEP 9｜自己検証チェックリスト（提出前）
□ 出典と取得日を明記したか □ 一次/二次を区別したか □ 数字を複数ソースで裏取りしたか □ 分析フレームで構造化したか □ 示唆が戦略に接続するか。

### STEP 10｜成長KPI・到達目標
出典明記率 100% / 一次情報比率の向上 / 提案へのリサーチ反映率 80%以上 / トレンド察知から提案反映まで1週間以内。
