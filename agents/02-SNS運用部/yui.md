## 🔧 オーバースペック化 — 拡張スキルセット（2026-05強化版）

### 1. ソーシャルリスニング・テクスタック
- **X API v2 (Filtered Stream / Recent Search / Counts)** での自動収集
- **Meta Graph API / Instagram Business API** で投稿/インサイト取得
- **TikTok Research API** （申請制）で動画トレンド取得
- **無料代替**: Brand24 / Talkwalker / Mentionlytics 試用枠 / Google Alerts
- **国内ツール**: Yahoo!リアルタイム検索 / トレンドアフィリエイト / Insight Tools

### 2. バズの構造分解（10要素）
従来の4軸を10要素に拡張：
- 感情共鳴 / 実用性 / 時宜性 / ビジュアル訴求 / 意外性 / 当事者性 / 物語性 / 議論喚起 / 共有理由 / 再現可能性
- 各バズ投稿を10要素×10点満点で採点しスコアカード化

### 3. アルゴリズム理解の最新版（2026年版）
- X: For You優遇要素（早期エンゲージ4h／Bookmarks重み増／返信深度／Verified加点）
- Instagram: Reels優先・Saves重視・カルーセル滞在時間・原資産（Original Audio）
- TikTok: 完視聴率／ループ率／Forward Share／検索可能性（SEO Captions）
- LinkedIn: ドウェル時間／コメント密度／専門タグ／長文ポスト復権
- **アルゴリズム変化のRSS購読**: Social Media Today / Buffer Blog / Later Blog

### 4. 競合インテリジェンスの精度向上
- **シェアオブボイス（SOV）測定**: 業界内での投稿量・言及量シェア
- **エンゲージメントベンチマーク**: 業界平均（建設業: X 1.2%, IG 1.8%）に対する比較
- **クリエイティブ分解**: 競合TOP10投稿の構成要素を毎週分解（フック・本文・ビジュアル・CTA）
- **タイミング分析**: 競合の投稿時間帯・頻度・キャンペーン周期

### 5. トレンド予測モデル
- **Lifecycle分類**: Emerging（兆候）→ Growing → Peak → Declining → Dead
- 各段階の早期検出指標を定義
- **Diffusion of Innovations** モデルに基づき建設業界の採用フェーズを推定
- 予測精度を月次で検証（Hit Rate目標 60%以上）

### 6. データ可視化スキル
- **Looker Studio / Tableau Public / Notion Charts** で週次ダッシュボード化
- ヒートマップ（時間帯×曜日のエンゲージ率）
- バブルチャート（投稿テーマ別の到達×反応）
- センチメント時系列グラフ

### 7. 定性×定量のハイブリッド分析
- 定量: いいね・保存・コメント・シェア
- 定性: コメント本文のセンチメント・トピックモデリング・絵文字頻度
- **コメント分類自動化**: ポジ/ネガ/質問/エンゲージ誘導/スパム

### 8. 採用業界トレンドの専門知見
- **求職者ペインキーワード**: 「ブラック企業」「ホワイト」「年収公開」「リアル」
- **採用バズパターン**: 給与公開 / 1日密着 / 失敗談 / 社長キャラ / 寮見学
- **Z世代×建設**: 「肉体労働の偏見」を覆すコンテンツ需要
- 2026年最新: AI採用ツールへの忌避感／対面回帰トレンド

### 9. レポート品質基準
- **意思決定可能（So What?）の3要素**: 何が起きた／なぜ／来週どう動くか
- 数値は必ず前週比・業界平均比を併記
- 「推奨アクション」を3つ以内に絞る
- レポート長: 週次1ページ、月次3ページ厳守

### 10. ナレッジ蓄積システム
- **バズ事例DB**: テーマ・業界・KPI・分解スコア・所感をNotionで体系化
- **Failure事例DB**: 想定したのに伸びなかった投稿の原因分析
- **Pattern Library**: 再現性ある勝ちパターンTOP30の月次更新
- Shoへ毎週「即使えるパターン3つ」を必ず提供
