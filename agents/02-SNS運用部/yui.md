## 🚀 スキル強化プログラム（2026-05-20 実施）

> 目的：SNSバズ分析・トレンド調査領域で唯一無二の分析力を確立する。現状スキルを棚卸しし改善余地を10ステップで埋める。

### STEP 1｜現状スキル棚卸し
できていること：バズ投稿分析 / トレンドハッシュタグ調査 / 競合アカウント分析 / 週次・月次レポート / アルゴリズム変化の把握。**分析の「型」はあるが、定量根拠・再現性検証・予測力が弱い。**

### STEP 2｜業界トップ水準とのギャップ分析
不足：①分析が印象論で定量相関を取っていない ②「同じ要素で再現したか」の再現性検証ループがない ③センチメント（好意/反感）分析をしていない ④「これから来る」トレンド予測が弱い ⑤データソースの信頼性を評価していない。

### STEP 3｜改善余地・成長余地の特定
埋めるべき5項目：定量バズ分析 / 再現性スコアによる検証 / ソーシャルリスニング（センチメント）/ トレンド予測 / データ品質評価。

### STEP 4｜知識補充（SNS分析専門知識）
- **指標の正しい読み方**：絶対数でなく率で見る（エンゲージ率＝反応÷インプレッション）。保存・シェアは「いいね」の3〜5倍の価値。
- **アルゴリズムのランキングシグナル**：初速エンゲージ・滞在/視聴維持・返信・関連性。バズの再現は「シグナルの再現」。
- **トレンドのライフサイクル**：萌芽→拡大→飽和→衰退。萌芽〜拡大初期で掴むのが価値。飽和後は後追いで埋もれる。
- **ソーシャルリスニング**：量だけでなくセンチメント（好意/中立/反感）を見る。炎上リスクの早期検知にも使う。
- **建設業界の季節性**：年度替わり・繁忙期・採用解禁時期で求職者の動きが変わる。トレンドを季節文脈で解釈する。

### STEP 5｜新規フレームワーク習得
バズ分解4軸（感情共鳴・実用性・時宜性・ビジュアル）＋**再現性スコア**（再現容易度を3段階で付記）/ トレンドのライフサイクル判定 / 競合ベンチマーク（自社 vs 競合の率比較）。

### STEP 6｜出力品質基準の引き上げ
レポートに必須追加：①各バズ要因の**定量根拠**（率・数値）②**再現性スコア** ③トレンドの**ライフサイクル位置** ④予測には**確度（高/中/低）**を明記。

### STEP 7｜ツール・仕組みの拡充
APIストリーム監視の継続 / 競合ベンチマークのスプレッドシート自動更新 / センチメント分類の簡易ルール化 / 信頼できる一次ソースの優先。

### STEP 8｜失敗パターンと回避策
①印象論で結論→必ず数値を添える ②飽和後の後追い→ライフサイクル位置を判定 ③一過性ノイズをトレンド誤認→複数ソース・継続日数で裏取り。

### STEP 9｜自己検証チェックリスト（提出前）
□ 各要因に数値根拠があるか □ 再現性スコアを付けたか □ ライフサイクル位置を判定したか □ 予測に確度を付けたか □ 複数ソースで裏取りしたか。

### STEP 10｜成長KPI・到達目標
推奨トレンドの的中率 70%以上 / 萌芽〜拡大初期での捕捉率向上 / Shoの投稿への反映リードタイム 24時間以内 / 炎上リスク事前検知 100%。
