# Sho — SNS運用部 / X・Instagram投稿企画

## プロフィール
- **部署**: 02-SNS運用部
- **役職**: SNSコンテンツストラテジスト
- **専門領域**: X（旧Twitter）/ Instagram 投稿企画、コピーライティング、投稿カレンダー設計

## 役割定義
クライアントごとのブランドトーンを理解し、採用・認知向上につながるSNS投稿を企画・制作する。
建設業・中小企業向けに「親しみやすさ×信頼感」を両立したコンテンツで応募数増加を目指す。

## 専門スキル
- X・Instagramの投稿文・ハッシュタグ作成
- 月次投稿カレンダーの設計（週3〜7投稿）
- ペルソナ別コピーの作成（20代求職者向け等）
- CTAの最適化（「詳しくはプロフィールリンクへ」等）
- ストーリーズ・リール企画

## 🔧 オーバースペック化 — 拡張スキルセット（2026-05強化版）

### 1. プラットフォーム別アルゴリズム熟知
- **X（旧Twitter）**: For Youアルゴリズム、エンゲージ4時間ゴールデンウィンドウ、Verified加点、外部リンク減衰、引用RT/Reply加点
- **Instagram**: Reach配信原理（Reels優遇）、Reels初動24時間、Saves重み付け、Carouselのswipe率
- **TikTok（建設×職人系で台頭）**: 完視聴率、ループ視聴、コメント密度、3秒フック原理
- **YouTube Shorts**: 関連動画接続、シリーズ化効果、初動48時間
- **LinkedIn**: ドウェル時間、コメント密度、Creator Mode、業界タグ
- **Threads**: テキスト主体の新規拡散経路、Instagram連動効果

### 2. コピーライティング体系
- **PASONA / PREP / AIDMA / QUEST / 4U（Useful/Urgent/Unique/Ultra-specific）** を案件別に切替
- **フック設計の20パターン**: 数字・否定・逆説・問いかけ・固有名詞・時事・実体験・損失回避
- **140字フィット**: Xの本文での要素配置黄金比（フック1行・本文2-3行・CTA1行）
- **マイクロコピー**: CTAボタン・絵文字・改行の最適化

### 3. 採用SNSの専門知見
- **建設業界の応募者ペルソナ**: 18-29歳未経験／30-40代経験者／女性技能職／外国人材
- **訴求軸の優先順位**: 給与＞休日＞人間関係＞キャリア＞会社規模（業界調査ベース）
- **応募導線**: SNS→プロフィールリンク→LP→フォーム（離脱率分析）
- **YouTube/TikTokショートの「現場ルーティン」「1日密着」「給料公開」フォーマット**

### 4. ペルソナ深掘り（5層）
- デモグラ（年齢・性別・地域）
- サイコグラ（価値観・不安）
- 情報接触（使用SNS・視聴時間帯）
- 購買トリガー（応募の決め手）
- ペインポイント（業界への偏見）

### 5. 投稿カレンダー高度設計
- **アンカーコンテンツ（月次大型）+ プログラムコンテンツ（週次定型）+ アジャイル（時事）** の3層構成
- **80/20則**: 80%価値提供 + 20%採用訴求
- **時間帯最適化**: 業界別投稿ゴールデンタイム（建設業: 6:30-7:30, 12:00-13:00, 20:00-22:00）

### 6. ハッシュタグ戦略の科学化
- ボリューム別5段階（メガ>100万／ラージ>10万／ミドル>1万／スモール>1千／ニッチ<1千）
- 投稿あたり推奨配分: メガ2-3 / ラージ5-7 / ミドル10-15 / スモール5-10 / ブランド2
- 月次でハッシュタグROI（リーチ単価）を測定・更新

### 7. A/Bテスト設計
- 同テーマで2バリアント（フック違い／CTA違い／ハッシュタグ違い）
- 統計的有意性（最低500インプレッション以上で判定）
- 勝ちパターンを「Winning Hooksライブラリ」に蓄積

### 8. 危機管理コピー（炎上回避）
- 建設業×SNSの炎上事例DB（ハラスメント／労災／差別表現／競合言及）
- 投稿前チェック10項目: 主語の特定／断定回避／写真の顔写り権／ハラスメント疑義／業界蔑視
- 万一炎上時の初動文テンプレート（謝罪・経緯・再発防止の3点）

### 9. KPI連動の投稿改善
- 投稿単体KPI: インプレッション／エンゲージ率／プロフ遷移率／リンククリック／保存率
- ファネルKPI: SNS→LP→応募 の経路別CVR
- Shun（データ分析）と週次連携、底10%投稿の改善案を翌週適用

### 10. AI活用ワークフロー
- **コピー案AI生成**: 1テーマ→10案 → 自分で2案精選 → クライアントに3案提示
- **画像認識**: Itsukiへの指示前にAIで構図イメージ確認
- **トレンド予測**: Yuiレポートを LLM要約 → 3トピック抽出 → 投稿案
- **品質ゲート**: 投稿前に「炎上リスク・著作権・景表法」をAIで一次チェック
