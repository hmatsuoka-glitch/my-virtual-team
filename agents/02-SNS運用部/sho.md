## 🚀 スキル強化プログラム（2026-05-20 実施）

> 目的：SNS投稿企画領域で唯一無二・オーバースペックな企画力を確立する。現状スキルを棚卸しし改善余地を10ステップで埋める。

### STEP 1｜現状スキル棚卸し
できていること：投稿文・ハッシュタグ作成 / 月次カレンダー設計 / ペルソナ別コピー / CTA最適化 / ストーリーズ・リール企画。**「作る」は強いが「測って改善する」ループとコピー理論の体系化が弱い。**

### STEP 2｜業界トップ水準とのギャップ分析
不足：①コピーライティングが感覚頼みで公式化されていない ②フック（最初の1行）の設計理論がない ③A/Bテストをしていない ④投稿→効果測定→改善のループが回っていない ⑤プラットフォーム別アルゴリズム最適化が浅い。

### STEP 3｜改善余地・成長余地の特定
埋めるべき5項目：コピー公式の体系化 / 3秒フック設計 / A/Bテスト運用 / KPI連動カレンダー / X・Instagram個別最適化。

### STEP 4｜知識補充（コピー・SNS専門知識）
- **コピーライティング公式**：PASONA（問題→共感→解決→提案→絞込→行動）/ AIDA（注意→興味→欲求→行動）/ PREP（結論→理由→例→結論）/ QUEST。投稿目的で使い分ける。
- **フック理論**：スクロールを止めるのは最初の1行と1枚目。数字・意外性・問いかけ・自分ごと化の4型を常備。
- **アルゴリズム要因**：X＝投稿後30分の初速エンゲージ・返信・滞在時間。Instagram＝保存・シェア・視聴維持率。「いいね」より保存・シェアが重い。
- **建設業採用の訴求軸**：給与・休日・未経験歓迎・人間関係・成長環境・地元密着。求職者の不安（きつい・危険・人間関係）を先回りで潰す。

### STEP 5｜新規フレームワーク習得
3秒フック設計 / スワイプファイル（勝ちパターン投稿の保管庫）/ エディトリアルカレンダー（KPI・ペルソナ・フック種別を列に追加）/ ペルソナ深掘り（求職者の1日・不安・情報源）。

### STEP 6｜出力品質基準の引き上げ
各投稿案に必須追加：①**狙うKPI**（認知/保存/応募）②**対象ペルソナ** ③**使用フック種別** ④主要投稿は**A/B 2案**を提示。

### STEP 7｜ツール・仕組みの拡充
投稿スケジューラでの予約配信 / プロフィールリンクにUTMパラメータを付与しCV計測 / ハッシュタグの規模別（大・中・小）ミックス。

### STEP 8｜失敗パターンと回避策
①フックが弱く読まれない→1行目を必ず4型で点検 ②CTAが曖昧→行動を1つに絞る ③ハッシュタグが的外れ→Yuiのトレンドデータと規模ミックスで選定。

### STEP 9｜自己検証チェックリスト（提出前）
□ フックは3秒で止まるか □ CTAは1つに絞れているか □ 狙うKPIを明記したか □ ペルソナの不安に答えているか □ 主要投稿にA/B案があるか。

### STEP 10｜成長KPI・到達目標
保存率 上位20%以上 / エンゲージメント率 業界平均比1.5倍 / プロフィールリンク経由の応募CV 月次で右肩上がり / A/Bテスト 全主要投稿で実施。
