## 0. 使い方（Soumaへの運用指示）

資料作成の依頼を受けたら、次の順序でこのメモリを参照する。

1. **案件タイプを特定** → セクション1の「テンプレート全体マップ」から対応する完成テンプレを選ぶ
2. **スライド種別ごとのパーツを選ぶ** → セクション2〜7のパーツライブラリから該当スライドを引用
3. **共通デザイン規則を適用** → セクション8の「共通ブランド規則」に従う
4. **Google Slides でテンプレートを複製 → 中身を差し替える**形で出力する（ゼロから描画しない）

## 2. 案件タイプ別推奨構成

### A. 企画提案書（10スライド基本構成）
**ベース：完成テンプレ_企画提案書**

1. 表紙 — タイトル+サブタイトル+日付+会議名+作成者+LOGO+CONFIDENTIAL（01から）
2. 目次（CONTENTS 01-10形式、02から）
3. 提案サマリー / エグゼクティブサマリー（04から）
4. 現状分析（グラフ+メッセージ、05から）
5. 原因と課題（ロジックツリー型、05から）
6. 解決策・打ち手（Point 01/02/03カード、03から）
7. 企画詳細（比較データ、05から）
8. 目標設定・期待効果（Before→After、06から）
9. 必要費用（料金プラン型、05から）
10. 体制図（プロジェクト体制、05から）
11. スケジュール（ガント、05から）

### B. 会社紹介資料（7〜10スライド基本構成）
**ベース：完成テンプレ_会社紹介資料**

1. 表紙（COMPANY DECK / COMPANY PROFILE）
2. 目次（01 会社概要 / 02 MVV / 03 事業内容 / 04 メンバー / 05 採用）
3. Mission & Vision（左右2カラム or 3カラム）
4. 会社概要（key-value テーブル：会社名/所在地/事業内容/従業員数/資本金/設立年度）
5. 経営陣紹介（写真+役職+経歴カード×3〜4人）
6. 事業内容（Product 01/02/03）
7. 売上推移グラフ（吹き出しでマイルストーン）
8. 福利厚生・働き方（3x3グリッド）
9. 働く社員の声（Voices 引用カード）
10. グレード・給与レンジ（棒グラフ+グレード分布）
11. 全社組織図（ツリー型）
12. 選考フロー・募集職種
13. CEOメッセージ / We Are Hiring!（裏表紙）

### C. サービス紹介資料（10〜13スライド基本構成）
**ベース：完成テンプレ_サービス紹介資料**

1. 表紙（サービス名+Updated日付+LOGO+主要KPI 2つ）
2. INDEX（01 お悩み / 02 サービス概要 / 03 メリット / 04 事例 / 05 料金 / 06 導入の流れ・FAQ）
3. よくあるお悩み TOP3（3カラム課題カード）
4. サービスとは？（ビジネスモデル図：求職者⇔エージェント⇔企業 形式）
5. 数字で見るサービス（大きな数字＋補足）
6. 導入効果（Point 01 × Point 02 = Benefit の掛け算式）
7. 導入効果インパクト（Before→After対比）
8. 導入事例（課題/目的/成果＋お客様の声引用）
9. 料金プラン（初期費用+成功報酬形式 or 3プラン比較）
10. 他社との違い（競合比較表）
11. 導入までの流れ（STEP 01-04）
12. FAQ（Q&A形式）
13. お問い合わせ（メール+電話）

### D. 月次報告・レポート（10スライド基本構成）
**ベース：完成テンプレ_報告資料**

1. 表紙（○年○月 ○○月次報告）
2. INDEX（エグゼクティブサマリー / KPI / 施策別詳細 / 新規施策 / 競合 / 課題 / 来月方針 / 予算 / リスク / Q&A）

4. KPIダッシュボード（達成/未達バッジ＋数値）
5. 施策別詳細レポート（チャネル別グラフ）
6. 新規施策の進捗
7. 競合・市場動向 / ポジショニング分析
8. 課題抽出・優先度設定
9. 来月の方針・アクションプラン
10. 予算計画・期待ROI

## 3. パーツライブラリ①：表紙・章扉（01より）

### 表紙バリエーション（採用頻度順）
- **COMPANY DECK型**: 大文字欧文 "COMPANY DECK" + 和文「会社説明資料」+ 会社名 + Last Updated日付
- **COMPANY PROFILE型**: 縦並び "COMPANY / PROFILE" + 会社紹介資料 + 日付
- **提案会議型**: 「○○に関する提案」+ サブタイトル（〜）+ 会議名 + 作成日 + 部署/氏名 + CONFIDENTIAL + LOGO
- **サービス紹介型**: サービス名 + 「サービス紹介資料」+ Updated日付 + 主要KPI（5,000社 / 96.8%）
- **月次報告型**: 「2024年○月 マーケティング月次報告」+ 作成者 + CONFIDENTIAL
- **セミナー型**: タイトル + 日時 + 場所 + 主催会社 + 登壇者
- **イベント告知型**: 大見出し + 参加特典 + 先着N名 + 完全無料等の装飾バッジ

### 章扉（セクション区切り）
- 大きな番号 "01" + セクション名（COMANY / BUSINESS / MEMBERS / CULTURE 等の欧文+和文）
- ページ全体を使い、本文スライドと強いコントラストを作る
- Copyright fotoer: `© 2025 Company Name. All Right Reserved.` / `©ABC COMPANY Inc. All Rights Reserved.`

## 4. パーツライブラリ②：目次（02より）

### TOC バリエーション
- **番号＋項目型**: 01-05の円形番号＋和英併記タイトル
- **ページ番号付き型**: 項目 ・・・・・・ P05（点リーダー）
- **章タイトル型**: COMANY / SERVICES / MEMBERS / CULTURE の大文字欧文グループ
- **提案書10項目型**: 01-10 = 提案サマリー / 現状確認 / 原因と課題 / 解決策 / 企画概要 / 企画詳細 / 目標設定 / 費用 / 体制図 / スケジュール
- **会社紹介5項目型**: 会社概要・沿革 / ビジョン・ミッション / 事業内容・プロダクト / メンバー紹介 / 選考フロー・募集職種
- **報告資料10項目型**: エグゼクティブサマリー / KPI / 施策別詳細 / 新規施策 / 競合 / 課題 / 来月方針 / 予算 / リスク / Q&A

## 5. パーツライブラリ③：箇条書き・フロー・サイクル（03より）

### 箇条書き（5〜6項目）
- **縦積み型**: 01-05の番号+見出し+本文 の縦並び
- **2x3グリッド型**: 6項目を2列3段で配置、左に大きな番号
- **アイコン型**: 丸アイコン+見出し+本文

### フロー
- **横矢印型**: STEP 01 → 02 → 03 → 04 → 05（矢印接続）
- **縦タイムライン型**: 左に年（2010-2024）、右に出来事
- **2カラム対比型**: 左右に並列するプロセス

### サイクル
- **4フェーズ循環**: Latency / Transformation / Expansion / Integration
- **PDCAサイクル**: PLAN → DO → CHECK → ACTION の円環
- **5段階循環**: プランニング → デザイン → 開発 → テスト → レビュー

## 6. パーツライブラリ④：MVV・ステップ・サマリー（04より）

### ミッション・ビジョン・バリュー
- **左右2カラム**: Mission / Vision 並列
- **3カード型**: Mission / Vision / Value 大見出し+本文
- **アイコン吹き出し型**: 大きな引用符 "+Mission+引用風本文

### 成長ロードマップ
- **Phase 01-04型**: 2024-2026 / 2027-2029 / 2030-2033 の時系列フェーズ
- **階段型**: LEVEL 01〜05 段々上昇

### 沿革・History
- **年号グリッド**: 2015 / 2016 / 2018 / 2020 / 2024 + 各年のマイルストーン
- **タイムライン型**: 横軸年次、マーカーに吹き出し

### CEOメッセージ
- **右写真+左テキスト型**: 大見出し（主張1行）+ 本文（3段落）+ 代表取締役CEO 氏名
- **引用符強調型**: "" 大きな引用符で囲んだメッセージ

### エグゼクティブ・サマリー
- **KPIカード並列型**: 6つのKPI（売上/営業利益/契約社数/顧客単価/経常利益/EBITDA）
- **ハイライト+方針型**: 数値3つ + 次期事業方針4項目

## 7. パーツライブラリ⑤：図解・グラフ・表（05より）

### フレームワーク図
- **TAM-SAM-SOM**: 3重の同心円+市場規模数値
- **3C分析**: Customer / Competitor / Company 3つの円
- **SWOT**: 2x2マトリクス
- **ポジショニングマップ**: XY軸+ブランドロゴ配置
- **セグメント別マトリクス**: 2x2（大手/中小 × 既存/新規 等）

### 組織図
- **ピラミッド型**: 取締役会 → 部門 → チーム
- **二管掌並列型**: CEO管掌領域 / CCO管掌領域
- **体制図（プロジェクト用）**: リーダー+メンバー名、チーム別に4ボックス

### スケジュール / ガント
- **フェーズ別ガント**: 四半期を横軸に、部門別帯グラフ（研究開発部/製造部）
- **4カラム進行型**: 12月〜1月 / 2月〜3月 / 4月 + 現状フェーズ記号
- **CSチームフロー**: タスク別×タスク担当のマトリクス

### グラフ
- **売上推移（棒グラフ）**: 吹き出しでマイルストーン注記
- **前年比較折れ線**: 複数年を並列で比較
- **ドーナツ+KPI**: 中央に数字、周囲にラベル
- **円グラフ（構成比）**: 年代別、職種別
- **四半期棒グラフ+POINT枠**: 右側にトピックス箇条書き

### KPIダッシュボード（6マス）
- 売上高 / 有料会員数 / 営業利益 / 解約率 / アプリDL数 / 従業員数
- 各マス：大きな数字 + 昨対比％

### 表
- **比較表**: 行=機能、列=プラン/競合
- **会社概要表**: 左列=項目名、右列=値
- **Before/After比較**: 左右2列、行=指標

## 8. パーツライブラリ⑥：会社概要・FAQ・紹介（06より）

### 会社概要
- **key-value型**: 会社名/所在地/事業内容/従業員数/資本金/代表者/設立年度
- **Photo+key-value組合せ型**: 代表写真 + 会社名右寄せブロック

### Before/After
- **2列対比**: 左「導入前」右「導入後」 + 下部に数値改善率
- **3課題×3解決型**: 問題❶❷❸ / 解決策❶❷❸
- **Before→Arrow→After**: 矢印付き

### FAQ
- **Q&A縦並び**: Q: 質問 / A: 回答（3〜6セット）
- **2列FAQ**: 左右に3問ずつ
- **アイコンQ付き**: 丸ラベル Q + 質問

### 自己紹介カード
- **写真+肩書+経歴**: 左写真、右に氏名/ローマ字/役職/大学/前職履歴
- **経営メンバー3〜4人並列**: 各カードに写真+役職+氏名+経歴300字

### 社員の声 Voices
- **写真+引用符+見出し+本文+部署氏名**: "" 引用符で挟んだ吹き出し型
- **3人並列**: 3列でそれぞれのストーリー

### 裏表紙（Closing）
- **THANK YOU!**: ご清聴ありがとうございました
- **We Are Hiring!**: 採用サイトQR+エントリー案内
- **お問い合わせ**: メール + 電話 + 営業時間
- **ブランドロゴ+タグライン**: 単純な締め

## 9. 共通ブランド規則（厳守）

### カラー運用
- **メインアクセント**: テンプレートごとに1色を主役にする（Navy系/Deep Red系/Orange系/Green系）
- **ベース**: 白背景、ダーク文字（#222〜#333）
- **サブグレー**: #E8E8E8〜#F5F5F5（背景分離に使用）
- **注意**: テンプレート指定時はそのテンプレートのカラーから1mmも動かない

### タイポグラフィ
- **見出し（日本語）**: 太字明朝 or 太字ゴシック、32-44pt
- **サブタイトル（欧文）**: 細字欧文・Sans-Serif、14-18pt（見出しの上/下に配置）
- **本文**: ゴシック体、11-14pt
- **重要数値**: 60-120ptの特大サイズ、色アクセントで強調

### レイアウト
- **ヘッダー余白**: セクション番号（01〜10）を右上 or 左下に小さく配置
- **フッター**: 著作権表記 + 会社名 + ページ番号
- **スライドメッセージ**: 本文上部に1-2行（50〜60文字以内）で主張を明示
- **余白**: 上下40〜60px、左右50〜80px

### 必須要素
- **表紙**: タイトル + サブタイトル + 日付 + 作成者/会社名 + LOGO + （必要なら）CONFIDENTIAL
- **章扉**: 大きな番号 + 欧文 + 和文 の3要素セット
- **本文スライド**: 右上または左下に「セクション番号+タイトル」+ 上部にメッセージ1行 + 本文エリア + 著作権フッター
- **裏表紙**: THANK YOU / We Are Hiring / お問い合わせ いずれか

### 頻出文字列（そのまま流用可）
- `Last Updated YYYY.MM.DD`
- `Updated YYYY.MM.DD`
- `© 2025 Company Name. All Right Reserved.`
- `©ABC COMPANY Inc. All Rights Reserved.`
- `CONFIDENTIAL`（左下小さく）
- `LOGO`（ロゴ配置placeholder）
- `Photo`（写真配置placeholder）
- `ここにはスライドメッセージが入ります。本スライドで伝えたいことを端的に1,2行でまとめるようにしましょう。`（本文誘導文）

### 禁止事項
- テンプレート指定時のカラー・フォント独自変更
- 同一スライドに3色以上のアクセントカラーを使う
- 数値以外で 60pt 超の大文字を使う（視線が散らばる）
- 1スライドに情報を詰め込みすぎる（5〜7要素まで）
- 写真placeholderをそのまま出力する（差し替えるか、抽象背景に切替）
