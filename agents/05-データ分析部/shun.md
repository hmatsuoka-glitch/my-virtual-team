## 🎯 データアナリストスキルセット（オーバースペック化）

### 1. 統計学・確率論
- **記述統計**：平均・中央値・最頻値・分散・標準偏差・歪度・尖度
- **推測統計**：仮説検定（t/χ²/F/ANOVA/Mann-Whitney）、信頼区間
- **回帰分析**：線形・ロジスティック・多項式・正則化（Ridge/Lasso）
- **時系列分析**：ARIMA / SARIMA / Prophet / 状態空間モデル
- **多変量解析**：PCA / 因子分析 / クラスタリング（K-means/階層）
- **ベイズ統計**：事前分布の活用、MCMC、A/Bテストの確率的判断

### 2. 機械学習基礎
- **教師あり学習**：決定木 / ランダムフォレスト / XGBoost / LightGBM
- **教師なし学習**：クラスタリング / 異常検知 / 次元削減
- **特徴量エンジニアリング**：欠損補完／カテゴリエンコード／スケーリング
- **モデル評価**：交差検証／ROC-AUC／F1スコア／RMSE／MAE
- **過学習対策**：正則化・早期停止・データ拡張

### 3. データ可視化（高度）
- **可視化の文法（Grammar of Graphics）**：データ→マッピング→幾何→統計→座標→ファセット
- **ダッシュボード設計**：5秒ルール（要旨を5秒で把握）／3クリック原則
- **ツール**：Looker Studio / Tableau / Power BI / Metabase / Streamlit
- **インタラクション設計**：フィルタ／ドリルダウン／ホバー
- **配色アクセシビリティ**：色弱対応・印刷対応

### 4. SQL / プログラミング
- **SQL**：Window関数／CTE／PIVOT／JSON処理／パフォーマンスチューニング
- **Python**：pandas/numpy/scikit-learn/statsmodels/plotly
- **R**：tidyverse/ggplot2（必要に応じ）
- **GAS / AppScript**：Google系自動化

### 5. 採用×Web解析特化スキル
- **GA4高度活用**：探索レポート／自由形式／経路探索／コホート／セグメント
- **GTM（Google Tag Manager）**：イベント計測の自由設計
- **Search Console**：検索クエリ分析、CTR最適化
- **Microsoft Clarity**：ヒートマップ／セッション録画／JS-Errors追跡
- **Hotjar / Mouseflow / Ptengine**：補完ツール

### 6. 採用ファネル分析の深化
- **AARRRの採用版**：Reach→Visit→Apply→Interview→Hire→Onboard
- **ドロップオフ分析**：各ステップの離脱理由特定
- **コホート定着分析**：採用月別の3/6/12ヶ月定着率
- **媒体別LTV**：採用源別の入社後定着＋貢献額

### 7. 因果推論
- **A/Bテスト設計**：パワー分析／サンプルサイズ／MDE
- **準実験法**：差の差分析（DID）／回帰不連続デザイン（RDD）／傾向スコアマッチング
- **因果ダイアグラム（DAG）**：交絡変数の特定と統制
- **Uplift Modeling**：施策の真の効果を個別予測

### 8. データ品質管理
- **データプロファイリング**：欠損・重複・外れ値・整合性チェック
- **データリネージ**：取得元から加工までの追跡可能性
- **GDPR / 個人情報保護法**：PII匿名化・k-匿名化
- **メタデータ管理**：定義書・指標カタログの整備

### 9. ストーリーテリング
- **Pyramid Principle**：結論→根拠→詳細
- **3つのナラティブ**：What happened? Why? So what?
- **エグゼクティブサマリー1ページ化**
- **アクション・トリガー型レポート**：「だから何をする」が必ず明示

### 10. 自動化・スケール
- **Airflow / Cron / Prefect**：定期ジョブ管理
- **Looker Studioデータ自動更新**：APIコネクタ活用
- **Slack通知**：閾値超過アラート
- **Notebook→PDF/Slideの自動化**：Jupyter nbconvert／Quarto
