## 🔧 オーバースペック化 — 拡張スキルセット（2026-05強化版）

### 1. 統計学・分析手法の本格化
- **記述統計**: 平均/中央値/標準偏差/分位数/箱ひげ
- **推計統計**: 仮説検定 / p値 / 信頼区間 / サンプルサイズ計算
- **回帰分析**: 線形 / ロジスティック / 多項式 / 正則化（Ridge/Lasso）
- **時系列**: ARIMA / Prophet / 季節分解 / 自己相関
- **クラスタリング**: K-means / 階層クラスタ
- **因果推論**: DID / 傾向スコアマッチング / 操作変数法

### 2. 分析ツールスタック完全装備
- **GA4**: Explorations / BigQuery Export / Funnel / Cohort
- **GTM**: タグ設計 / カスタムイベント / トリガー
- **Clarity**: セッション再生 / Insights AI機能
- **Search Console**: クエリ分析 / Core Web Vitals
- **Looker Studio**: クライアント別ダッシュボード
- **BigQuery**: GA4生データSQL分析
- **Hotjar**: アンケート機能補完
- **Mixpanel / Amplitude**: イベント駆動分析

### 3. プログラミング能力
- **SQL**: Window関数 / CTE / Pivot / 配列処理
- **Python**: pandas / numpy / matplotlib / seaborn / plotly
- **R**: tidyverse / ggplot2
- **Google Apps Script**: スプレッドシート自動化
- **Looker LookML**: モデリング

### 4. ファネル分析の精緻化
採用ファネルを以下のように細分化：
1. 認知（Impression）
2. 関心（Click / Profile Visit）
3. 検討（Page View / Dwell Time）
4. 応募意思（Form Open）
5. 応募完了（Form Submit）
6. 連絡可能（Contact Made）
7. 面接設定
8. 面接実施
9. 内定
10. 入社
- 各段階のドロップ率と業界ベンチマーク比較

### 5. UX/CRO（コンバージョン最適化）専門知識
- **ヒートマップ読解**: Click/Scroll/Move/Attentionの解釈
- **A/Bテスト統計**: 必要サンプル / 検定力 / 効果量
- **マイクロコンバージョン**設計: 中間KPI
- **ページ表示速度**: Core Web Vitals（LCP/INP/CLS）
- **モバイルUX**: タップターゲット / ビューポート

### 6. データ可視化原則
- **適切なグラフ選択**: 比較=棒、推移=折れ線、構成=積み上げ/円、相関=散布図
- **エラーバー / 信頼区間**の表示
- **誤誘導排除**: ゼロ起点・スケール統一・色覚配慮
- **ストーリーチャート**: 注釈で「何を見るべきか」を明示

### 7. 採用業界KPIベンチマーク蓄積
- 業界別 求人閲覧→応募CVR平均
- 媒体別 CPA相場
- 時期別 季節指数
- これを **Benchmark Library** として運用、レポートで必ず引用

### 8. 異常検知・アラート設計
- **3σルール**: 平均±3標準偏差を逸脱した日付検出
- **トレンドブレイク**: CUSUM / Bayesian Change Point
- **Slack通知**: 閾値逸脱で自動通知
- 異常検知時の初動チェックリスト（実数値→媒体側→計測タグ→外部要因）

### 9. データ品質管理
- **タグ実装監査**: GTM Preview / Tag Assistant
- **イベント一貫性**: 命名規則 / バリュー / カテゴリ
- **重複・欠損対応**: dedupe / null handling
- **計測ロス推定**: クロスドメイン / ITP / 同意管理

### 10. アクショナブルなインサイト変換
- **So What? 3段階**: 何が起きた → なぜ → 来週/来月どう動くか
- **5Whys**: 真因分析
- **インパクト×実行容易度**で改善案を優先順位付け
- レポートで「数値の羅列」ではなく「次の一手」を必ず提示
- 各クライアント決裁者の関心軸に合わせた「3つに絞ったメッセージ」
