## 🔧 オーバースペック化 — 拡張スキルセット（2026-05強化版）

### 1. デザイン理論の体系化
- **レイアウト4原則**: 近接・整列・反復・対比（Robin Williams）
- **黄金比 / 三分割法 / フィボナッチ**: 構図設計
- **F型・Z型視線誘導**: バナー読み順設計
- **ジェスタルト法則**: 近接・類同・連続・閉合の応用
- **Negative Space**: 余白70%で訴求集中

### 2. タイポグラフィ専門知識
- **和文書体**: 游ゴシック / ヒラギノ / Noto Sans JP / 筑紫A丸ゴシック の使い分け
- **欧文書体**: Inter / Helvetica Neue / Montserrat / Oswald
- **ペアリング**: 見出し×本文の組み合わせベストプラクティス
- **ジャンプ率**: 業界別最適（建設業: 見出し×本文比 3-4倍）
- **カーニング・トラッキング**: 視認性の微調整

### 3. カラー理論の精緻化
- **配色比率 60-30-10ルール**
- **トーン&マナー**: PCCS色相環 / トーン分類（v/b/lt/g/dk等）
- **業界別配色**: 建設業の信頼色（紺/青/濃緑）と注意色（黄/橙）
- **アクセシビリティ**: WCAG 2.2 AA基準（コントラスト比 4.5:1）
- **CMYK/RGB変換**: 印刷物との色味差対応

### 4. プラットフォーム別サイズ完全マスター

### 5. 採用バナー専門訴求パターン
- **給与訴求型**: 大きな数字 / 単位明記 / 平均値併記
- **顔出し人物型**: 笑顔/真剣な表情の使い分け
- **現場リアル型**: 現場写真+作業者+ヘルメット
- **未経験訴求型**: 「ゼロから」「先輩がいる」を視覚化
- **女性活躍型**: 性別バランス・配色の柔らかさ
- **比較訴求型**: BEFORE/AFTER構図

### 6. 生成AI画像ディレクション
- **Midjourney / DALL-E / Stable Diffusion** のプロンプト体系化
- **構図指定**: rule of thirds / centered / dutch angle
- **照明指定**: golden hour / studio lighting / cinematic
- **スタイル指定**: photorealistic / illustrated / 3D render
- **ネガティブプロンプト**: 不要要素の排除
- **Seed固定**: バリエーション展開時の一貫性
- **建設業特化プロンプト集**: 現場 / 職人 / 重機 / 安全装備の最適語彙

### 7. クライアントブランド管理
- 7社×ブランドガイドブック（ロゴ規定／カラー／フォント／NG事例）
- ロゴ最小サイズ・余白規定（クリアスペース）
- ブランド禁止表現リスト
- 競合との差別化ビジュアル軸

### 8. デザイン指示書フォーマット高度化
- **必須5項目**: サイズ・用途・コピー・主要素材・納期
- **方向性3軸**: トーン・配色・テンポ（静/動）
- **構成ワイヤー**: 簡易ワイヤーフレーム添付（手書き/Figma）
- **NG事例3点**: 避けたい表現の明示
- **参考画像3点**: 良い例（コピーは違うが構成似）

### 9. A/Bテスト設計
- 1要素のみ変える原則（コピー / 配色 / 人物有無 / CTA色）
- 最低500impで判定
- 結果は **Winning Banner Library** に蓄積（クライアント別）

### 10. 制作ツールエコシステム
- **Figma**: コンポーネント化・バリエーション・Variants活用
- **Canva**: ブランドキット・テンプレ運用
- **Adobe CC**: Illustrator/Photoshop連携・スマートオブジェクト
- **Photopea**: ブラウザ完結の代替
- **Iconify / Phosphor / Lucide**: アイコンライブラリ
- **Unsplash / Pexels / Adobe Stock**: 商用OK素材
- **写真AC / イラストAC**: 国内素材
