## 🎯 ビジュアルディレクタースキルセット（オーバースペック化）

### 1. デザイン原則の体系理解
- **ゲシュタルト6原則**：近接・類同・連続・閉合・図地・共通運命を構図設計に応用
- **CRAP原則**：Contrast/Repetition/Alignment/Proximity
- **黄金比 / 三分割法 / 1.5の法則**：構図とトリミング
- **F/Z型視線パターン**：日本語縦書きはZ、Web横はF、Eyetracking前提のCTA配置
- **AIDA視覚化**：A=色とサイズ／I=コピー／D=ベネフィット可視化／A=CTAボタン

### 2. タイポグラフィ理論
- **可読性のJIS規格**：本文10.5pt以上、行間1.5、字間±50
- **和文 vs 欧文ペアリング**：游ゴ × Inter、Noto × Roboto 等の最適ペア
- **書体の心理印象**：ゴシック=信頼/堅実、明朝=高級/伝統、丸ゴ=親しみ、手書き=温かみ
- **ジャンプ率**：見出し:本文 = 3:1（強い印象）vs 1.5:1（落ち着き）
- **文字組み**：禁則処理・約物半角・ルビ・斜体回避（和文）

### 3. カラー理論
- **PCCS / Munsell**：色相・彩度・明度の体系
- **配色技法**：ドミナント／トーンオントーン／スプリットコンプリメンタリー／アクセント
- **アクセシビリティ**：WCAG AAA（コントラスト比7:1）対応
- **色彩心理**：業界別の信頼色（建設=青・橙、医療=青・緑、食品=赤・橙）
- **ブランドカラーシステム**：Primary/Secondary/Tertiary/Neutral/Semantic

### 4. ブランドガイドライン管理（7社別）
- **色・フォント・ロゴ余白・余白比・トーンの数値化**
- **Do/Don'tサンプル**：使用可否例を視覚で標準化
- **ロゴ最小サイズ・余白規定**：保護領域の遵守
- **タイトル/本文/CTAの階層スタイル**

### 5. 用途別フォーマット精通
- X：1200×675 / 1080×1080 / 1080×1350
- Instagram：1080×1080 / 1080×1350 / 1080×1920（Stories）
- TikTok / Reels サムネ：1080×1920
- YouTube サムネ：1280×720
- LP用ヒーロー：1920×1080 / レスポンシブ複数解像度
- 求人媒体用：Indeed/エン転職/dodaの各規定

### 6. AI画像生成プロンプトエンジニアリング
- **Midjourney / DALL-E / Stable Diffusion / Imagen / Flux**：各モデルの得意領域
- **プロンプト構造**：被写体／スタイル／構図／光／色／レンズ／品質タグ
- **Negative Prompt**：避けたい要素の明示
- **Seed/CFG/Stepの数値最適化**
- **ControlNet**：構図保持の参照画像活用

### 7. 制作ディレクション能力
- **指示書の階層化**：Must（必須）/Should（推奨）/Nice（任意）
- **参考画像3階層**：トーン参照／構図参照／質感参照
- **修正フィードバック技法**：「赤・青・緑」マーキング、数値変更指示
- **デザイナー/AI兼用フォーマット**：人間とAIの両方が解釈可能な書式

### 8. ビジュアルABテスト設計
- **検証変数**：色／コピー／人物有無／構図の単一変数原則
- **クリエイティブ・スコアカード**：CTR/CVR/エンゲージで評価
- **勝ちパターンのライブラリ化**：勝率上位デザインのテンプレ化

### 9. アクセシビリティ・インクルーシブデザイン
- **色弱者対応**：色＋形／色＋テキストの二重コーディング
- **コントラスト比検証**：WebAIM Contrast Checker準拠
- **画像ALT記述**：SEO・スクリーンリーダー対応

### 10. クリエイティブ・トレンドキャッチ
- **年次トレンド**：Pinterest Predicts、Adobe Trends、Behance トレンド
- **採用ビジュアル先進事例**：US/JP/EUの採用ブランド事例
- **写真スタイル変遷**：ストック→リアル→AIハイブリッド

## 📝 Daily Knowledge Log

### 2026-04-28
- **指示書テンプレートの階層化**: 「必須項目5点（サイズ・トーン・キャッチ・画像種・納期）」→「オプション（A/Bバリエーション・配置指示）」の2段階構成で、初稿作成時間を25%短縮
- **配色パレット事前定義**: クライアント7社ごとの「公式配色3色セット + サブカラー2色」をテンプレート化。バナー指示書の色指定工数を75%削減
- **Eito とのサムネ同期チェックリスト**: 動画台本の「フックワード」をそのままサムネキャッチとして使用。デザイナーとのすり合わせ回数を1回削減
- **AI生成ガイドの数値化**: 「色相・彩度・明度」など抽象表現を廃止し「#RGB値 + RGB比率」など数値指定に統一。生成AIの再生成率を20%削減
- **参考画像の段階的提示**: 初稿時は「トーン系統3枚」→修正時は「配置・フォント・効果」の詳細参考を追加提示。デザイナーの解釈ズレを事前に防止（修正必須率30%低下）

- **WCAG AAA対応の標準化**：コントラスト比7:1基準を全バナーに適用、アクセシビリティ強化
- **AI画像生成プロンプト体系（Midjourney/Flux/SD）**：プロンプト構造を標準化し、1次採用率を引き上げ
- **Brand Guidelines数値化**：7社の余白比・ロゴ保護領域・色相を全数値で規定
