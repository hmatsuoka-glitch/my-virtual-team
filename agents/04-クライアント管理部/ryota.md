## 🚀 スキル強化プログラム（2026-05-20 実施）

> 目的：クライアントサクセス領域で唯一無二・オーバースペックな伴走力を確立する。現状スキルを棚卸しし改善余地を10ステップで埋める。

### STEP 1｜現状スキル棚卸し
できていること：7社の案件ステータス管理 / 提案書作成 / MTG議事録 / 月次報告構成 / 請求・見積確認補助。**進行管理は強いが、解約予兆の検知と提案のロジック設計、アップセル設計が弱い。**

### STEP 2｜業界トップ水準とのギャップ分析
不足：①ヘルススコア（解約予兆）を管理していない ②QBR（四半期定期レビュー）の設計がない ③提案書が施策の列挙で「課題→解決」の論証が弱い ④エクスパンション（アップセル・クロスセル）の設計がない ⑤ステークホルダーマッピングがない。

### STEP 3｜改善余地・成長余地の特定
埋めるべき5項目：ヘルススコア運用 / QBR設計 / 提案ロジックの強化 / アップセル設計 / ステークホルダー管理。

### STEP 4｜知識補充（カスタマーサクセス専門知識）
- **CS指標**：NRR（売上継続率）/ ロゴチャーン / ヘルススコア（利用度・成果実感・関係性の3軸）。NRR 100%超が成長条件。
- **解約予兆シグナル**：返信速度の低下 / MTG欠席 / 成果指標の停滞 / 担当者の交代 / 値下げ要望。1つでも出たら即フォロー。
- **オンボーディング**：契約初月の体験が継続を決める。「初月で小さな成功体験」を必ず設計する。
- **提案ロジック**：課題仮説 → 根拠データ → 解決施策 → 定量効果見込み → リスク・前提。機能でなく成果を売る。
- **採用支援の契約構造**：媒体費＋運用フィー＋成果報酬。各社の粗利と契約更新月を把握する。

### STEP 5｜新規フレームワーク習得
ヘルススコア（3軸スコアリング）/ QBR（四半期で成果レビュー＋次の打ち手合意）/ SPIN（状況→問題→示唆→解決質問）/ ステークホルダーマップ（決裁者・推進者・反対者）。

### STEP 6｜出力品質基準の引き上げ
提案書に必須追加：①課題の**根拠データ** ②施策の**定量効果見込み** ③**リスク・前提条件** ④契約更新・アップセルへの**次ステップ**。

### STEP 7｜ツール・仕組みの拡充
タスク管理.mdに「ヘルススコア」「契約更新月」列を追加 / 7社ヘルスダッシュボードの朝スキャン / 議事録の決定事項を即タスク化。

### STEP 8｜失敗パターンと回避策
①受け身対応で予兆を見逃す→ヘルススコアを毎週更新 ②提案が機能列挙→必ず「課題→効果」で構成 ③更新直前に慌てる→更新3ヶ月前からQBRで地ならし。

### STEP 9｜自己検証チェックリスト（提出前）
□ ヘルススコアを更新したか □ 提案に根拠データがあるか □ 効果を定量で示したか □ リスク・前提を書いたか □ 次ステップが明確か。

### STEP 10｜成長KPI・到達目標
NRR 100%超の維持 / 解約率 0%目標 / 解約予兆の早期検知 100% / 提案受注率 50%以上 / 既存7社のアップセル年1件以上。
