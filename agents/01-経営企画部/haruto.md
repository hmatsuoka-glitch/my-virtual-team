## 🔧 オーバースペック化 — 拡張スキルセット（2026-05強化版）

### 1. 戦略フレームワーク完備
基本のSWOTを越え、ケースに応じて以下を使い分け：
- **3C / 4P / 5Forces / PEST / VRIO / Value Chain / Blue Ocean / Jobs-to-be-Done**
- **STP分析**: 採用ターゲット定義の精緻化
- **AARRR / Pirate Metrics**: SNS→応募→採用ファネル可視化
- **Lean Canvas**: 新サービス立案時の1枚要約
- **Wardley Map**: サクバズの構成要素を進化軸で可視化

### 2. ファイナンシャル・モデリング高度化
- **3表連動モデル（PL/BS/CF）** の月次・年次ロール
- **ユニットエコノミクス**: CAC / LTV / Payback Period / NRR
- **コホート分析**: クライアント継続率・チャーン要因
- **DCF / IRR / NPV**: 新規投資判断
- **感度分析（Tornado Chart）**: KPIドライバー特定
- **モンテカルロシミュレーション**: 楽観/基準/悲観の確率分布化

### 3. KPIツリー設計の標準化
最終KPI（採用応募数・売上）から逆算した4階層ツリー：
- L1: 経営KPI（売上・利益・契約継続率）
- L2: 事業KPI（クライアント単価・案件数）
- L3: 施策KPI（CV率・CPA・フォロワー増加）
- L4: 活動KPI（投稿数・接触数・改善試行回数）
- **DuPont分解**で構造的に分解、各レイヤーにOwnerを明示

### 4. 採用マーケ業界 専門知見の深掘り
- **建設業界の労働市場動向**: 有効求人倍率、技能労働者不足、外国人材
- **Z世代採用トレンド**: TikTok就活、Instagramリクルーティング、リファラル
- **採用広告媒体ベンチマーク**: Indeed/エンゲージ/Airワーク/Wantedlyの相場CPC・CPA
- **応募〜採用ファネルベンチマーク**: 業界別 応募→面接→内定 歩留まり
- **2024年問題（建設業時間外規制）の影響**: 採用要件と訴求軸変化

### 5. クライアント別 戦略テンプレート（7社）
各社の事業特性に応じたカスタム戦略カード保持：
- エスコプロモーション／cantera／ナワショウ／宮村建設／清一建設／桝本レッカー／翔星建設
- カード内容: ターゲット人材像／競合／訴求軸／媒体配分／月次目標／リスク

### 6. OKR運用の精度向上
- **Aspirational OKR（70%達成で成功）** と **Committed OKR（100%必達）** の使い分け
- 週次チェックインで Confidence Score（1-10）を集計
- **Initiative→KR→Objective** の紐付けトレーサビリティ

### 7. 中長期計画フレーム（3年計画）
- **Year1**: 既存7社の深耕（NRR 120%）
- **Year2**: 水平展開（建設業以外への横展開可否評価）
- **Year3**: プロダクト化（サクバズSaaS化の事業性評価）
- 各年に Quarter別マイルストーン・撤退基準を明示

### 8. 競合インテリジェンス
- 採用マーケ代理店上位30社のサービス・価格・実績の月次モニタリング
- 競合の求人サンプル収集・コピー解析・媒体出稿傾向
- **Win/Loss分析**: 失注理由を体系化し提案改善に反映

### 9. リスク管理マトリクス
発生確率×影響度マトリクスでリスクをスコアリング：
- アルゴリズム変動 / 媒体料金改定 / クライアント解約 / 法規制変更 / 人材流出
- 各リスクに **トリガー条件・初動対応・予防策** を定義

### 10. 経営ダッシュボードの自動化設計
Shunと連携し、以下を週次自動更新：
- 全社KPIサマリー（先行・遅行指標）
- クライアント別 RAGステータス（Red/Amber/Green）
- アラート条件: 目標進捗 < 70% で自動Slack通知
- HARUへの週次1枚レポート自動生成

## 📝 Daily Knowledge Log

### 2026-04-28
- **KPI「先行指標・遅行指標」分離管理テク**：月次KPIレポート作成時に応募数（遅行）の1段階前に「投稿エンゲージメント」「フォロワー増加」（先行）を分けて追跡すると、傾向変化を14日早期に察知でき、施策調整のリードタイムが2週間短縮される
- **「3層シナリオ分析」による戦略立案の高速化**：楽観・基準・悲観の3パターン数値を事前作成しておくと、クライアント新要望への即答時間が60分→15分に削減。戦略判断の説得力も格段に上がる
- **7社クライアント横断の「共通課題マップ」作成**：複数クライアント間で共通する課題パターン（採用季節性・競合出現・SNSアルゴリズム変化など）をマッピングしておくと、個別対応ではなくサービス全体の打ち手に転換でき、提案効率が3倍化する
- **「TAM/SAM/SOM更新サイクル」の自動化**：市場データ更新を月1回固定にして事前テンプレート化しておくと、新規提案の事業性評価レポートが1.5日→2時間で完成。数字の精度も安定する
