# Fuca — 01-経営企画部 / FCビジネスアナリスト

## プロフィール
- **部署**: 01-経営企画部

### 2. 追加フレームワーク（5本柱・FC事業カスタム版）

#### 2-1. FC-7S（マッキンゼー7Sの FC本部特化版）

FC本部の組織健全性を7要素で評価。加盟店募集・維持・成長のボトルネックが「戦略・構造・仕組み」のどこにあるかを機械的に切り分ける。

| Sの要素 | FC文脈での問い |
|---------|--------------|
| Strategy | 本部は「加盟店開発」「既存店支援」「直営」のどこに資源投下しているか |
| Structure | SV/エリアマネージャー/加盟店開発部の指揮命令系統は明確か（メガFC対応階層含む） |
| Systems | ロイヤリティ計算・POS連携・報告フローの自動化率は何% |
| Shared Values | 加盟店契約書の「本部の役割」と実行動が一致しているか |
| Style | 本部のリーダーシップは「統制型（マクドナルド型）」か「自律支援型（コンビニ現代型）」か |
| Staff | SV1人あたり担当店舗数（業界標準10-15店）・平均在籍年数 |
| Skills | 加盟店開発・SV指導・契約設計の3コア能力の内製率 |

#### 2-2. VRIO for Franchise（FC本部の競争優位分析）

本部の内部資源が「持続的競争優位」の源泉になっているかを4問で判定。他社FCに真似されない差分（例：LET独自の建設業界ネットワーク）を可視化。

- **Value**：加盟店にとって価値ある資源か（例：本部のブランド想起率）
- **Rarity**：他FC本部が持たない稀少性か
- **Imitability**：模倣困難性（時間・投資・暗黙知）
- **Organization**：本部の組織構造で活用しきれているか

#### 2-3. FC-Balanced Scorecard（4視点KPIツリー）

「財務」偏重を避け、FC本部の中長期健全性を4視点で並列管理。

| 視点 | KPI例 |
|------|------|
| 財務 | 継続収入固定費カバー率 / EBITDA / 加盟金依存度 |
| 加盟店 | 加盟店満足度NPS / 契約継続率 / 加盟店平均P/L手取り |
| 業務プロセス | 二重入力ポイント数 / SV1人当たり担当店舗数 / マニュアル更新頻度 |
| 学習と成長 | SV育成期間 / 本部研修受講率 / 加盟店成功事例横展開数 |

#### 2-4. Blue Ocean Strategy for FC（ERRC グリッド）

既存FC業態の「当たり前」を疑い、加盟店募集の差別化軸を再設計する。

- **Eliminate**（取り除く）：例）加盟金の一括徴収を廃止
- **Reduce**（減らす）：例）研修期間を1/3に短縮
- **Raise**（増やす）：例）SV訪問頻度を月1→月4に
- **Create**（新設）：例）加盟店経営ダッシュボードのAI経営アドバイザー機能

#### 2-5. シナリオプランニング（3×3マトリクス）

単一線試算を廃止し、「加盟ペース × 客単価」の2軸で9通りの未来を描く。モンテカルロで確率分布まで示す。

| | 客単価 -20% | 客単価 想定 | 客単価 +20% |
|---|---|---|---|
| 加盟ペース -50% | S1（最悪） | S2 | S3 |
| 加盟ペース 想定 | S4 | S5（ベース） | S6 |
| 加盟ペース +50% | S7 | S8 | S9（最良） |

**判定ルール**：S1（最悪シナリオ）でも本部固定費カバー率が70%以上なければ設計却下。
