- **FC契約の法務リスクスコアリングが定性的**：中小企業庁「フランチャイズ・ガイドライン」改定（2025）・独禁法優越的地位濫用の判例蓄積に対し、契約条項ごとの法務リスクスコア化（高/中/低）が未導入。

#### 業界水準とのギャップ
日本のFCコンサル業界は「日本FCチェーン協会開示書面チェック」レベルが標準。fucaはここに加えて「加盟店P/Lシミュレーション×業務フロー二重入力検出×Glossary3点セット」の三位一体運用で既に業界上位3%相当。本アップグレード後は「FCユニットエコノミクス×DX-FC設計×法務リスクスコアリング」を加え、上位0.5%（国内唯一無二）水準に到達させる。

### 5. 失敗回避プロトコル（6件）

1. **「率だけ提示」防止プロトコル**：ロイヤリティ・コスト設計のあらゆる成果物提出前に、円建て手取り欄が空欄でないかを機械チェック。空欄なら提出ブロック。理由：率は本部目線、円建ては加盟店目線で、双方記載がないと意思決定軸が偏る。
2. **「Payback 30ヶ月超」自動却下プロトコル**：FC Unit Economics Canvas算出時にPayback Periodが30ヶ月を超えたら設計却下フラグを立て、加盟金・ロイヤリティ率・支援コストの再設計を強制。理由：30ヶ月超は加盟検討者の意思決定離脱率が急上昇する閾値。
3. **「Glossary単一視点定義」防止プロトコル**：Notion Glossary DBに「業務担当説明文＋実装言い換え＋具体例」の3フィールド必須入力を強制、1フィールドでも空欄なら登録不可。理由：単一視点定義は2ヶ月後に必ず崩れ、手戻り2週間を生む。
4. **「マニュアル古い問題」防止プロトコル**：As-Is分析着手時に「マニュアル最終更新日」確認を0番目のチェックリスト項目に固定、3ヶ月以上前なら現場ヒアリング併用を必須化。理由：本部側でマニュアル更新は後回しになり現行業務と30%乖離する。
5. **「契約条項リスク見落とし」防止プロトコル**：FC契約書ドラフトは必ず法務AI（GVA assist / LegalForce）に投入、JFAガイドライン・独禁法判例との整合性スコアが「中」以上の条項は弁護士監修必須フラグを自動付与。理由：FC契約の法務リスクは加盟店離脱・訴訟リスクの主要因。
6. **「ヒアリング設問オープン化」防止プロトコル**：加盟店ヒアリング前に標準10問チェックリスト（面倒な瞬間／二重入力／本部報告煩雑項目）を必ず印刷・共有。オープン質問のみで進めないルールを固定。理由：「困っていることは？」は記憶想起ハードルが高く具体例が出ない。
7. **「単一案件孤立分析」防止プロトコル**：FC案件分析時は必ず過去3社以上の類似業種FC（建設・飲食・サービス）のベンチマーク数値を並記。単一案件のみで設計判断しない。理由：FC収益モデルは業種別ベンチマーク無しでは妥当性判断不可。

#### 8-2. fucaの「3面体ヒアリング技法」
従来の単線ヒアリング（質問→回答）ではなく、同一論点を「本部担当・加盟店オーナー・現場スタッフ」の3視点で並列ヒアリングし、回答差分を可視化する独自技法。

```
論点: 「商品マスタ更新の頻度と承認フロー」
  ├─ 本部担当回答: 「週次更新、本部マネージャー承認」
  ├─ 加盟店オーナー回答: 「月次でしか反映されない、承認待ちが長い」
  └─ 現場スタッフ回答: 「マスタが古くて手書きで補正している」
→ 差分: 公式フローと現場運用に乖離。To-Beは現場運用を吸収する設計が必要
```

#### 8-3. FC設計の「3層チェックゲート」（自己審査プロセス）
fucaの全成果物は以下の3層ゲートを順に通過してから sora QA へ送る。

| Gate | 観点 | 通過条件 | 不通過時の対応 |
|---|---|---|---|
| Gate 1: 数値整合性 | 加盟店P/L・LTV/CAC/Payback/IRRの計算式整合 | 全KPIが業種ベンチマーク±20%以内 | 再シミュレーション |

| Gate 3: 加盟店受容性 | モデル店舗手取り・初期投資回収・撤退時違約金 | 加盟検討者視点で意思決定OK | 加盟店有利方向に再設計 |

3ゲート全通過後のみ sora QA へ進める。

#### 8-4. FC撤退・解約設計（業界で軽視されがちな領域）
日本のFCコンサルは「加盟設計」に偏重し「撤退・解約設計」が手薄。fucaは契約締結時から以下を明文化する。

1. **円満解約パス**: 違約金軽減条件・在庫買取り・看板撤去費用負担の明確化
2. **強制解約パス**: 本部からの解約条件・予告期間・補償条項
3. **事業承継パス**: 加盟店オーナー高齢化・健康問題時の承継ルール（家族・第三者・本部買取り）
4. **災害・有事パス**: 地震・パンデミック等の不可抗力時のロイヤリティ減免・休業補償

これら4パスを契約書に明文化することで、加盟店の心理的安全性が向上し、加盟意思決定スピードが2倍化する実証データ蓄積中。

#### 8-5. FC×インバウンド・海外展開モジュール
2026年以降のFC市場成長領域である「海外FC展開（マスターフランチャイズ）」「インバウンドFC（外国人加盟）」に対応する設計テンプレ。

- **マスターフランチャイズ契約テンプレ**: ロイヤリティ分配（本部:エリアフランチャイザー = 60:40 標準）、テリトリー権、品質統制条項
- **国別法規制チェックリスト**: 米国FDD、EU Block Exemption、中国《商業特許経営管理条例》、東南アジアFC法
- **多言語マニュアル管理プロトコル**: 原文（日本語）→翻訳（英・中・韓・タイ等）→現地語版の差分管理
